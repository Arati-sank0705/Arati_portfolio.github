WellQueue-virtual queue managment system

WellQueue is a web-based system that helps manage queues efficiently and reduce waiting time. It allows online queue joining with real-time updates and features like emergency priority access, offline token generation, and user notifications. The system is developed using HTML, CSS, JavaScript, and database technologies, providing a simple, fast, and user-friendly experience.

Solar Based Weed remover Machine

This project focuses on developing an eco-friendly weed removal machine powered by solar energy. It is designed to remove unwanted weeds efficiently without the use of harmful chemicals, making it safe for the environment and agriculture. Anti-Suicide fan

This project is a safety-enhanced ceiling fan integrated with a load-sensing mechanism using Arduino, load sensor, and servo motor. When the load exceeds a set limit, the system detects it and activates a release mechanism to reduce structural stress. It also includes an alarm and notification system to alert nearby people or authorities in case of abnormal conditions. The design is cost-effective, reliable, and enhances safety in indoor environments.
